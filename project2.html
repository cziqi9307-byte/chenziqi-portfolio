情感交互装置

通过面部识别技术和LED矩阵实现的情感表达艺术装置

Computer Vision OpenCV Arduino LED Matrix Python

项目概述

这是一个融合艺术与科技的交互式装置作品，通过计算机视觉技术实时捕捉观众的面部表情，分析情绪状态，并通过大型LED矩阵以光影和色彩的形式进行可视化呈现。

装置创造了一种人与科技之间的诗意对话，让观众能够直观地看到自己的情绪变化，探索情感表达的多种可能性。每个观众与装置的互动都是独一无二的艺术体验。

核心功能

  • 实时情绪识别

    基于深度学习的面部表情分析，识别喜悦、悲伤、惊讶等7种基本情绪

  • 动态视觉反馈

    根据情绪类型和强度生成独特的色彩组合和动画效果

  • 多人互动

    支持同时识别多个观众，创造集体情绪的可视化呈现

  • 数据记录

    记录互动数据，生成情绪变化趋势图和统计报告

技术实现

视觉识别

OpenCV + Dlib人脸检测，CNN情绪分类模型

显示系统

WS2812B LED矩阵，Arduino控制，自定义驱动

视觉设计

情绪-色彩映射算法，粒子系统动画效果
